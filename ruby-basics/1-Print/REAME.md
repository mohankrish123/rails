### How to print?
#### Prints with a new line
```
puts "Hello World!"
```
#### Prints with a new line and returns
```
p "Hellow World!"
```
#### Prints without a new line
```
printf "Hello World!"
```

#### Get input string
```
printf ("What's your name: ")
name = gets()
puts ("Hello #{name}")
```

#### Convert number to string
```
number = 1
str = "This number is: "

puts str + number.to_s
```
